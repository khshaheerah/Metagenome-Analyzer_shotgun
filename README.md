A **lightweight, self-contained R pipeline** for analyzing shotgun metagenomics sequencing data on personal computers without HPC access. Perfect for MacBook Air users, students learning metagenomics, and researchers with limited computing resources.

## 📋 Overview

This tool provides a complete workflow for shotgun metagenomics analysis including:

- ✅ **No internet required** - Fully self-contained with simulated data
- ✅ **Runs on any laptop** - Optimized for 8GB RAM, <2 minutes runtime
- ✅ **Publication-ready plots** - High-resolution (300 DPI) figures
- ✅ **Complete statistics** - Alpha/beta diversity, differential abundance, FDR correction
- ✅ **Educational** - Well-commented code perfect for learning

## 🚀 Quick Start

### Installation

```bash
# Clone the repository
git clone https://github.com/YOUR_USERNAME/shotgun-metagenomics-R.git
cd shotgun-metagenomics-R

# Run the analysis
Rscript metagenomics_analysis.R
